# Privacy Policy for ChatGPT DOM Trimmer

Last updated: [DATE]

ChatGPT DOM Trimmer is a Chrome extension that helps improve the responsiveness of long ChatGPT web conversations by collapsing older conversation turns into placeholders.

## What the extension does

The extension works only on ChatGPT web pages, including:

- https://chatgpt.com
- https://chat.openai.com

It modifies the page DOM in your browser to hide older visible conversation turns and restore them on demand.

## Data the extension accesses

The extension can access content on supported ChatGPT tabs because it needs to identify conversation turns and modify the page DOM for its core functionality.

The extension also stores user preferences locally in the browser, such as:

- keep visible turns
- restore batch size
- auto-collapse setting

## Data collection and transmission

ChatGPT DOM Trimmer does not send your chat content, personal data, or browsing data to any remote server controlled by the developer.

ChatGPT DOM Trimmer does not use analytics, tracking scripts, ads, or third-party telemetry.

ChatGPT DOM Trimmer does not sell, transfer, or share your personal data with third parties.

## Local storage

User settings are stored locally using Chrome's extension storage APIs.

## Remote code

ChatGPT DOM Trimmer does not load or execute remote code.

## Data retention

The extension stores only the settings needed for functionality, and those settings remain in your browser until you change them or remove the extension.
